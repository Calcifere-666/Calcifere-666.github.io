Étape 2 : Prompts

2.1

Pour la personnalisation du prompt on regarde les config de mon prompt personnel avec :echo $PS1 Nous avons donc toutes les informations sur la personnalisation de notre prompt :
[\033[0;31m]\342\224\214\342\224\200$([[ $? != 0 ]] && echo "[[\033[0;31m]\342\234\227[\033[0;37m]]\342\224\200")[[\033[0;39m]\u[\033[01;33m]@[\033[01;96m]\h[\033[0;31m]]\342\224\200[[\033[0;32m]\w[\033[0;31m]]
[\033[0;31m]\342\224\224\342\224\200\342\224\200\342\225\274 [\033[0m][\e[01;33m]$[\e[0m]

2.2

Expliquations en détails ci-dessous \[\033[0;31m\] : Définit la couleur rouge \342\224\214 et \342\224\200 : Ce sont des caractères Unicode (peut-être des barres verticales/horizontales pour structurer votre prompt).

2.3

Affichage d'une croix rouge si la dernière commande a échoué : $([[ $? != 0 ]] && echo "[\[\033[0;31m\]\342\234\227\[\033[0;37m\]]\342\224\200")
$([[ $? != 0 ]] && ...) : Vérifie si la dernière commande s'est terminée avec un code de sortie non nul (!= 0).
\342\234\227 : Représente un caractère Unicode en forme de croix (✗).
\[\033[0;31m\] : Affiche la croix en rouge.
\[\033[0;37m\] : Passe ensuite à la couleur blanche pour le reste du texte.

2.4

Affichage du nom d'utilisateur et de la machine : [\[\033[0;39m\]\u\[\033[01;33m\]@\[\033[01;96m\]\h\[\033[0;31m\]]
\[\033[0;39m\]\u : Nom de l'utilisateur en couleur par défaut.
\[\033[01;33m\]@\[\033[01;96m\]\h : Séparateur @ en jaune suivi du nom de l'hôte en cyan clair.
\[\033[0;31m\] : Retour à la couleur rouge pour fermer les crochets.

2.5

Affichage du répertoire courant : [\[\033[0;32m\]\w\[\033[0;31m\]]
\[\033[0;32m\]\w : Affiche le chemin courant (\w) en vert.
\[\033[0;31m\] : Retourne à la couleur rouge.

2.6

Nouvelle ligne et structure finale : \n\[\033[0;31m\]\342\224\224\342\224\200\342\224\200\342\225\274 \[\033[0m\]\[\e[01;33m\]\$\[\e[0m\]
\n : Ajoute une nouvelle ligne après les informations.
\[\033[0;31m\]\342\224\224\342\224\200\342\224\200\342\225\274 : Caractères Unicode pour dessiner une ligne ou un séparateur.
\[\e[0m\] : Réinitialise les styles à la fin.
